<b>MONITORAMENTO INTELIGENTE DE TEMPERATURA E UMIDADE PARA VINÍCOLAS</b>

Um projeto inovador voltado ao monitoramento de temperatura e umidade em vinícolas, ajudando produtores a preservar a qualidade dos vinhos e reduzir perdas. A tecnologia empregada permite uma análise precisa das condições ambientais, garantindo que os vinhos sejam armazenados nas condições ideais. Com o uso estratégico de sensores especializados, o sistema alerta os responsáveis sobre variações críticas, permitindo ações corretivas imediatas para evitar danos aos produtos.

Além da preservação da qualidade, o Lofhel contribui para a sustentabilidade do setor vinícola e para a rentabilidade dos negócios. A implementação dessa solução tecnológica assegura um gerenciamento mais inteligente dos estoques, promovendo produtos de maior valor agregado e reduzindo desperdícios financeiros.

## 🚩 Funcionalidades

### 📊 Dashboards
- **Dashboard individual de cada armazém:**  
Estatísticas sobre o estado de cada armazém, fornecendo dados de temperatura, umidade e situação dos sensores, com atualização contínua.

- **Dashboard da situação geral:**  
Estatísticas gerais dos armazéns, incluindo a quantidade de alertas na semana atual e na anterior, número de armazéns em estado de alerta e histórico de alertas.

### 🌌 Gerenciamento 
- **Funcionários:**  
O usuário pode cadastrar, atualizar e remover funcionários da empresa, desde que tenha permissão para isso.

- **Cargos:**  
O usuário pode cadastrar, atualizar e remover cargos da empresa. Cada empresa começa com o cargo de Representante Legal, podendo personalizar conforme as necessidades, definindo permissões e nomes dos cargos.

- **Armazéns:**  
O usuário pode cadastrar, atualizar e remover armazéns da empresa, fornecendo um nome e o grupo do vinho para que as métricas sejam corretamente aplicadas.

### 🌌 Suporte  
- **Bob IA:**  
Nosso agente de suporte N3 utiliza a inteligência artificial Bob IA para agilizar os processos e aumentar sua produtividade.

- **Jira:**  
Sistema de central de atendimento.

### 🌌 Outras Funcionalidades
- Login e cadastro.  
- Calculadora Financeira  
- Manual de instalação dos sensores

## 🚀 Instalação e Execução

### Pré-requisitos:
- Node.js  
- Navegador atualizado  
- MySQL configurado  
- Arduino IDE

### 🔧 Instalação:
1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/shadow-slave-project
   cd shadow-slave-project
2. Instale as dependências do web data viz dentro da pasta LOFHEL:
   ```bash
    npm install

3. Configure o banco de dados (arquivo .sql disponível na pasta /database).

4. Crie um arquivo .env na raiz do projeto com as seguintes informações:
   ```ini
    DB_HOST=localhost
    DB_USER=seu_usuario
    DB_PASSWORD=sua_senha
    DB_DATABASE=nome_do_banco
    MINHA_CHAVE=sua_chave_gemini 

    APP_PORT=3333
    APP_HOST=localhost
5. Execute o servidor:
   ```bash
    npm start
6. Abra o arquivo index.html no seu navegador para iniciar o site.
